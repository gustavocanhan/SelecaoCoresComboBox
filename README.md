# Seleção de Cores com Panel – Windows Forms

## Descrição
Aplicação Windows Forms desenvolvida em C# para praticar o uso dos componentes ComboBox, Panel, Label e Button, permitindo a seleção de cores e a alteração dinâmica da interface.

## Objetivo do exercício
- Aprender a usar o componente Panel
- Trabalhar com ComboBox e seleção de itens
- Manipular cores via código
- Praticar boas práticas de nomenclatura
- Organizar código em métodos

## Componentes utilizados
- Label
- ComboBox
- Panel
- Button

## Funcionalidades
- Carregamento dinâmico das cores no ComboBox
- Seleção de uma cor pelo usuário
- Alteração da cor de fundo do Panel
- Exibição do nome da cor selecionada
- Reset do formulário
